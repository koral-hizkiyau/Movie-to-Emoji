## Installation

To install this library, run:

```bash
$ pip install openai
$ pip install ttkbootstrap
$ pip install threading
$ pip install tkinter
$ pip install emoji
$ pip install time 
```
Go to https://openai.com/api/ and create an account.<br/>
Create your own KEY and update the app,using the menu bar - "Update Key"
